A) Quick start guide :

Javascript Code:

Cats Slider uses the jQuery framework. Load jQuery and the Cats Slider javascript files in the proper order. 

<script src="js/jquery-1.7.2.min.js"></script>
<script src="js/CatsSlider.js"></script>
and , 
function(){
	$('#slider').CatsSlider({
	width:800,
	height:480,
	delay:3000,
	animationDelay:1000,
	navigate:{enable:true,enableColor:"darkred",disableColor:"black",borderColor:'#aaaaaa'},
	next:"#next",
	prev:"#prev",
	autoSlide:true,
});

HTML Code :

<div id="slider">
	<button id='prev'>prev</button>
	<button id='next'>next</button>

</div>

B) Details description:

Parameter Name Value Description
width integer To indicate slider width
height integer To indicate slider height
delay integer ( mili second ) To indicate delay for every slide
animationDelay integer ( mili second ) Every Animation time
next css selector next button id or class or any css selector
prev css selector next button id or class or any css selector
current positive number to point the starting point of image to slide . first image number is 1
onChange function WOW slider genarate a event for every slide with index number as current
onhover "pause" or "play" on hover slide will pause for "pause"
autoSlide true or false for true image will slide after every delay time
navigate enable true or false true for visulize bottom navigator
enableColor color current bottom bar color
disableColor color all bottom bar color without current
borderColor color bar separator color
loader enable true or false to enable top loader
color color top loader color
animationStyle 'circle','rectangular','row','column','vclose','hclose' you can choose your fevarite animation from them

C) Layer Animation :

write a html code like- <img src="images/image5.jpg" /> 
<img src="images/image1.jpg" data-text-id="wow_text1" />

and ,

<div id="wow_text1">
	<div style="font-size:42px;color:black;width:400px;" animation-style="{left:900px;top:150px;delay:0px;opacity:.1;},{left:300px;top:150px;opacity:1;delay:600;}">
		6 Different Animation
	</div>		
</div>
here animation-style attribute will do every animation on your slide - {left:900px;top:150px;opacity:.1;delay:0;},{left:300px;top:150px;opacity:1;delay:600;} . any single animation collapse with {} and multi animation separated by ',' . All is css style without delay . delay indicate the duration to change layer style from current to given style . you can see demo file for better understanding .
